# AI Presentation Generator Agent

An AI-powered system that automatically generates professional presentations from a single topic using Groq API. It creates structured slides including title, outline, and detailed content with an interactive UI.

---

## Features

- AI-generated presentation from topic input  
- Automatic title and outline creation  
- Clickable outline to generate detailed slides  
- Professional slide structure (like real PPTs)  
- Font customization option  
- Dark/Light theme support  
- Export presentation option (PDF/PPT)  
- Modern and clean UI design  

---

## Tech Stack

- Python  
- Flask / FastAPI  
- HTML, CSS, JavaScript  
- Groq API (LLM)  
- JSON-based slide engine
